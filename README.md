<h1 align="center"><a href="
  shanote.onrender.com " target="_blank">ShaNote</a></h1>

A simple note taking app build using Laravel framework. Along with Laravel, TailwindCSS as CSS framework and Apline.js as JavaScript framework is used.

<p align="center">------- This Web Application is under development. -------</p>

This web application features,

- Full user authentication.
- Create, Read, Update and Delete notes.
- AI helper for writing notes.
- Public and private notes with Public note feed.
- Save public notes.
- Update profile details.
